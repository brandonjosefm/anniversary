# Imágenes

La experiencia actual no depende de fotografías (usa tipografía, emoji, gradientes y
formas CSS/SVG para mantenerse ligera y con carga instantánea). Esta carpeta queda
reservada por si luego quieres añadir, por ejemplo, una foto real en la pantalla final.
Si agregas imágenes, recuerda usar `loading="lazy"` y formatos comprimidos (WebP/AVIF).
